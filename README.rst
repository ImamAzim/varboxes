varboxes
===================

A varbox is an object to store permanently objects between python session. It uses json module, so it restricts the type of variable to store accordingely.


Installation
============

.. code-block:: bash

    pip install varboxes

Usage
=====


This is an example:

.. code-block:: python

    from varboxes import VarBox
    vb = VarBox('MyApp')
